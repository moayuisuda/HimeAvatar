# uni-shop

The backend of project-mononoke

## create venv

`python -m venv ./_venv`
`python3 -m venv ./_venv`

## active venv

windows: `.\_venv\Scripts\activate`
mac: `source ./_venv/bin/activate`

## install venv packages

`pip install -r ./requirements.txt`

## start server

`python src/server.py`
